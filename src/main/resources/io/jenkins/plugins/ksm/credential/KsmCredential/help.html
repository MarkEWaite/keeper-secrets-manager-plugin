Allows for storage of Keeper Secrets Manager credentials used for environmental injection of secret into your build. These credentials can be selected in the build and in pipeline workflow by their credential id. The values in these fields can be obtained from using the Keeper Secrets Manager CLI, or from the config of other Keeper Secrets Manager applications.
Using Commander, generate a one time access token to your application. It's important not to lock the IP since you are most likely going to be using the token on another machine. My Vault> secrets-manager client add --app MyApplication ------------------------------------------------------------------ One Time Access Token: XXXXXXXXXXX ------------------------------------------------------------------
